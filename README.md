A customizable random password generator built using HTML, CSS, and Vanilla JavaScript.
Generate secure passwords instantly with flexible length control and character customization options.
🚀 Features
Generate two random passwords at once
Set custom password length (max 30 characters)
Toggle:
Numbers
Symbols
Click password to copy to clipboard
Visual confirmation when copied
Clean and modern UI
🛠 Technologies Used
HTML5
CSS3
Vanilla JavaScript (No frameworks or libraries)
⚙️ How It Works
Passwords are generated using combinations of:
Uppercase letters
Lowercase letters
Numbers
Symbols
Character sets update dynamically based on toggle selections.
Password length defaults to 15 if no value is entered.
Clicking a generated password copies it using the Clipboard API.
